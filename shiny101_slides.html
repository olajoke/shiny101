Structure of a Shiny App

  • A single script (app.R) or two-file (ui.R and server.R)

  • Three major parts : The ui object, the server function and a call to ShinyApp
    # The definition of ui
    ui <- ...
    # The definition of server
    server <- ...
    # call to the app
    shinyApp(ui = ui, server = server)

Run a Shiny App

  • Click on “Run App”, if you are using the Rstudio

  • runApp(“newdir”)

  • “Ctrl + Shift + Enter”

  • “shiny::runApp()”
